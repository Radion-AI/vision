About this project

Vision is created as part of the assignment provided by The school of AI. This project uses flask framework and docker container
View on GitHub

Vision is inspired from Pytorch-Flask-Starter

By default, this app uses MobileNetV2 image classifier that was pre-trained on the ImageNet dataset. This can be easily changed with any custom deep learning model. If you wish to contribute to the project, please feel free to add more example models. More explanation as to how to contribute can be dound in the GitHub repository link.
